HTML Structure

The template is based on Skeleton Framework. It comes with two different width, 960px and 1200px. Check out our online demo to chose which one do you want to use.

If you want to use 960px width include skeleton.css file

If you want to use 1200px width include skeleton-wide.css file

CSS Structure

The template css files are separated in few smaller files so you can manage them better.

Javascript Structure

Amara use few jQuery scripts for the effects. All scripts are included in head part of the each html file. All functions that you want to customize are located in custom.js file.

Font and Theme Colors

The template use one separated css files for all theme colors. Chose one (only one ) of the 5 available colors and include it in to your template.

Font Family

Amara use Google Web Fonts to display fonts in template. You can chose any of 600+ web fonts at http://www.google.com/webfonts and embeded them inside the template.

You will also need to change font family inside style.css file

				h1, h2, h3, h4, h5, h6 {
				font-family: 'Oswald', sans-serif;
				font-weight: 400;
				line-height: 1.5; }
				body {
				font: 14px/20px 'Open Sans', sans-serif;
				font-weight: 300;
				color: #444;
				background-color: #fff;
				-webkit-font-smoothing: antialiased; /* Fix for webkit rendering */
				-webkit-text-size-adjust: 100%; }

Twitter Feed

Twitter feed use Seaofclouds with Tweet.js-Mod in order to display tweets. Since Twitter upgraded their API to 1.1 you will need to create application to get your consumer key, consumer secret, user token and user secret. Here is a quick video of how to create your application. Once you get your API keys go to js/twetter/index.php file and update the file with your keys.

Contact Form and Map

Contact Form

Update contact form with your email address. Open send.php file and update "$adminemail" with your email address.

Google Map

Google map is iframed. If you want to change it, go to google maps (classic version), enter your location, click on link button and pick up the iframed version of the link. Update your google map by replacing only the src path inside index.html file. Leave id="map" inside iframe so map could be responsive.
